# Pre-requisites
- Install [Node.js](https://nodejs.org/en/)

# Getting started
- Clone the repository
```
git clone  https://github.com/Rasmus-adn/jsscripts.git
```
- Install dependencies
```
cd <project_name>
npm install

- Add .env file
```
cd <project root>
touch .env
paste https://webhook.site/<key>

- Run project
```
npm start

- Run tests
```
npm test
